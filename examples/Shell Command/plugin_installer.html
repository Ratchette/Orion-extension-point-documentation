[Plugin Name] Example Shell Command Plugin

[Describe what your plugin does] The orion.shell.command service is used to contribute commands to the Shell page. When the service is executed, an object containing the user-supplied argument values is passed to the service's callback method. The command then optionally returns a response (a return value) to be displayed in the Shell page.
This particular example adds a command to the shell that translates a number from 0 to 9 into its english name

For more information regarding this extension point please see Eclipse Orion's Developer Guide

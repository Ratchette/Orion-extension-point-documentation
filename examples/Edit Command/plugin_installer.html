[Name of your plugin] Example Editor Command Plugin

[Explain what your plugin does] This page and its associated resources contain a basic plugin for Orion's Edit Command.
This particular example is of a basic command and generates C style comments for javascript files.
The command extension has two primary features.

  1. To change what text is selected in the editor
  2. To modify the contents of a selection

The edit command extension point can also be used to spawn a delegating UI, but this feature is not currently orion 4.0 compliant

For more information please see Eclipse Orion's Developer Guide.

[Plugin Name]Example Editor Outliner Plugin

[Explain what your plugin does here] This page and its associated resources contain a basic plugin for Orion's Outliner.

The outliner is a bar that hovers to the left of the file currently being viewed in the editor. The outliner can contain a tree of links to lines to particular lines in the file or other URLs.

For more information please see Eclipse Orion's Developer Guide.

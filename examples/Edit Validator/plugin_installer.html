[Plugin Name] Example Editor Validator Plugin

[Describe what your plugin does] An orion.edit.validator service provides a function that can check the contents of a file and return a data structure indicating where syntactic or semantic problems lie. The result of this service is used by the Orion UI to create annotations in the ruler beside each problematic line, and also to underline the specific portion of the document where the problem occurs.
This example will validate HTML files for best CSS practices. It produces an error if a tag contains inline styling, and a warning for every table tag.

[Plugin Name] Example Navigator Command Plugin

[Describe what your plugin does] The orion.navigate.command service is used to contribute commands to the actions (gear) menu that are relevant to a selected file or folder. When the service is executed, a file object or array of file objects is passed to the service's run method. The command can perform some operation on the provided files. If the command is simply linking to another page, a uriTemplate can be used to specify the link.
This example performs a google search on the selected file's name

For more information regarding this extension point please see Eclipse Orion's Developer Guide

[Plugin Name] Example Editor Outliner Plugin

[Describe what your plugin does] The outliner is a bar that hovers to the left of the file currently being viewed in the editor. The outliner can contain a tree of links to lines to particular lines in the file or other URLs.
This particular example generates a nested outline of media-wiki style headings (= heading =) from textfiles

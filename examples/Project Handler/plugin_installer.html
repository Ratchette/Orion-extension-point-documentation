[Plugin Name] Example Project Handler Plugin

[Describe what your plugin does] The orion.project.handler service is used to provide a custom project handler. Creating a custom project handler allows you to:

  • Create a project of a given type
  • Add dependencies of a given type
  • Add aditional informaion to project page
This particular example generates a nested outline of media-wiki style headings (= heading =) from textfiles

For more information regarding this extension point please see Eclipse Orion's Developer Guide

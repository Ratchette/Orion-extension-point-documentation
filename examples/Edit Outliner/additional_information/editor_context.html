Orion Editor Context Object information

Orion's Editor Context object can be used to explicitly update the content of Orion's Editor from inside your plugin

For the most recent description of the Editor Context object please see Eclipse Orion's Developer Guide

Methods

name
getAST(options)
description
Resolves to Object. Returns an AST (Abstract Syntax Tree) for the file being edited, if an AST provider is available. As of Orion 4.0 the only AST provider shipped with Orion is Esprima for JavaScript files. Additional providers can be contributed by plugins.
name
getCaretOffset()
description
Resolves to Number. Returns the offset of the editing caret.
name
getSelection()
description
Resolves to Selection. Returns the editor's current selection.
name
getText(start?, end?)
description
Resolves to String. Returns the text in the given range.
name
setCaretOffset(offset, show?)
description
Resolves to undefined. Sets the caret offset. If show is true, the editor will scroll to the new caret position.
name
setSelection(selection)
description
Resolves to undefined. Sets the editor's selection.

description

[Plugin Name] Example Shell Type Plugin

[Describe what your plugin does] The orion.shell.type service is used to contribute parameter types to the Shell page. These enable contributed commands to have custom parameters with dynamically-computed completion suggestions. The service is executed when the Shell page needs either completion suggestion values to display, or the string representation for a given value.
This particular example creates a shell type that represents a planet in our solar system. It comes with a breif orion.shell.command plugin that displays the previously created type for demonstration purposes only.

For more information regarding this extension point please see Eclipse Orion's Developer Guide
